# PTO IR 运算指南

本节记录 PTO Level-1/Level-2 IR 契约中的非 ISA 运算，这些运算不属于 `docs/isa/` 下的指令条目。

## 范围

- View 构造与切分运算。
- Tile 分配与地址移动运算。
- 核参数查询运算。
- Tile/标量取值与写值辅助运算。
- Level-2 可用的同步原语。
