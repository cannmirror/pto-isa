# TFILLPAD_EXPAND

## 指令示意图

![TFILLPAD_EXPAND tile operation](../figures/isa/TFILLPAD_EXPAND.svg)

## 简介

填充/填充时允许目标大于源。

## 数学语义

除非另有说明, semantics are defined over the valid region and target-dependent behavior is marked as implementation-defined.


### IR Level 2（DPS）
